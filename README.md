# GCP Cloud Functions

Simplest set up for running GCP Cloud Functions on local
environments

## Quick Start

```sh
⚡ git clone repo
⚡ npm i
# ⚡ yarn install
⚡ npm run start # <- start development env
⚡ npm run serve # <- start functions-framework
```

### FWIW

- Default port is `8080` and
- Signature type is `http`
